# Tal Goods — Homepage

Premium homepage built with TanStack Start + Vite + Tailwind v4 + Framer Motion.

## Local development

```bash
npm install
npm run dev
```

Open http://localhost:8080

## Production build

```bash
npm run build
```

Outputs Vercel-ready files under `.vercel/output` (Nitro `vercel` preset).

## Deploy to Vercel

1. Push this folder to a GitHub repo.
2. Import the repo in Vercel.
3. Framework Preset: **Other** (auto-detected).
4. Build Command: `npm run build`
5. Install Command: `npm install`
6. Output is handled automatically via Vercel Build Output API.

No environment variables required for the frontend-only homepage.
